Menggambar model

Menggambar model pada canvas yang tersedia dapat dilakukan dengan menggunakan 3 cara, yaitu:

1. Melakukan import file JSON

Anda bisa menggunakan canvas yang sebelumnya pernah anda buat di website ini dengan melakukan import file bertipe JSON

- Tekan tombol "Choose File", pilih file
- Tekan import untuk menampilkan gambar yang sudah pernah dibuat sebelumnya

2. Menggunakan shortcut button dan mouse

Dengan menggunakan cara ini, anda perlu menekan salah satu tombol line/ rectangle/polygon

- Untuk menggambar garis Anda cukup menekan 2 titik pada canvas, garis akan tergambar
- Untuk menggambar persegi, setelah menekan rectangle anda diminta memasukan ukuran persegi.
Setelah itu tekan pada 1 titik di canvas. Persegi akan tergambar
- Untuk polygon, setelah memilih pilihan menu anda akan diminta memasukan jumlah titik pada poligon
Setelah itu tekan kanvas sebanyak titik yang tadi telah dimasukan

3. Melalui configuration panel

Dengan menggunakan cara ini, bisa memilih warna dari objek yang digambar

- Mengisi jumlah vertics ( bisa ditambah dikurang dengan panah )
- Mengisi vertices pada kolom yang tersedia, vertices juga dapat diisi secara random dengan menekan "randomize"
- Jika titik koordinat simpul sudah sesuai, tekan render

Save dan Download

Jika anda ingin menyimpan canvas yang telah anda gambar. Sangat mudah, dengan menekan tombol save, dibagian "import & save"
